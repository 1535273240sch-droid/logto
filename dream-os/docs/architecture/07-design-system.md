# 07 - Design System

## 设计理念

**Living Interface**: 界面不是预先设计好的，而是随着用户目标自然生长。

## 色彩系统

- 背景: `#0a0a0f` (深黑) / `#fafafa` (浅白)
- 次级背景: `#12121a`
- 卡片背景: `rgba(255,255,255,0.03)`
- 边框: `rgba(255,255,255,0.08)`
- 主文字: `#e8e8ed`
- 次文字: `#888899`
- 弱文字: `#555566`
- 强调色: `#6c5ce7` (紫)
- 渐变: `linear-gradient(135deg, #6c5ce7, #a29bfe)`
- 成功: `#00b894`
- 错误: `#e17055`
- 警告: `#fdcb6e`

## 字体

- 系统字体栈: `-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif`
- 品牌名: 18px, 700 weight
- 文案: 14px, 400 weight
- 标题: 16px, 600 weight

## 间距

- 组件内间距: 8px / 12px / 16px
- 页面边距: 20px
- 黄金分割点: 输入框在屏幕 38% 位置

## 动画

- 过渡: `all 0.2s`
- Agent 生长间隔: 300ms
- Dream Field 背景: 10-20s 循环
- 输入框光晕: 0.3s 渐变

## 圆角

- 按钮: 8px
- 卡片: 12px
- 模态框: 16px
- 输入框: 10px

## 玻璃拟态

- 背景: `rgba(255,255,255,0.04)`
- 边框: `rgba(255,255,255,0.1)`
- 模糊: `backdrop-filter: blur(10px)`
